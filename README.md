# AndroidDecompiler

#How to Run this ?

Right Click on AndroidDecompiler 
        ---> 
           Run As --> 
                    Run Configurations 
                             ---> 
                                Java Application
                                   --> 
                                     Main 
                                       ---> Project --> AndroidDecompiler
                                       ---> Main Class --> com.googlecode.dex2jar.v3.Main
                                      Arguments 
                                       ---> ProgramArguments --> testapp.apk (name of apk file)
                                       
                                       Run :)


Output of this build is "testapp_dex2jar.jar"
